It is an open source scripting language that is widely used. PHP file can contain HTML, JavaScript, CSS, and PHP code. Scripts are executed on the server and returned to the browser as plain HTML.

PHP was developed by Rasmus Lerdorf in 1994 who wrote the number if CGI a binaries to maintain his personal homepage. PHP is a tool for making a dynamic content, interactive web pages or used for CGI scripts, it can also create, read, write and delete a files on the server, can add, delete and modify a data to the database and can encrypt data

PHP can generate dynamic page content, create, open, write,delete and close file on the server. It can add, delete, modify data in your database. It can send and receive cookies and can encrypt data.PHP originally known as Personal Home Page but it is now a recursive acronym and some they called it 'Perl Killer' of the server side scripting world. It is integrated with a number of popular databases, including MySQL, PostgreSQL, Oracle, Sybase, Informix, and Microsoft SQL Server. PHP is pleasingly zippy in its execution, especially when compiled as an Apache module on the Unix side. The MySQL server, once started, executes even very complex queries with huge result sets in record-setting time.

PHP supports a large number of major protocols such as POP3, IMAP, and LDAP. PHP4 added support for Java and distributed object architectures (COM and CORBA), making n-tier development a possibility for the first time. PHP has sufficient enough to run a largest social network and on different platforms and it is also suitable in most servers that we used today like apache, IIS and etc.

1. session_abort - it will disregard the array changes and finishes the session.

2. session_cache_expire - it will return the current cache expire

3. session_cache_limiter - it will either get and/or set the current cache limiter

4. session_commit - It is the alias of session_write_close

5. session_create_id - it will create a new session id

6. session_decode - it will decode the session data from a session encoded string

7. session_destroy - it will destroy all data which is registered to a specific session

8. session_encode - it will encode the current session data as a session encoded string

9. session_gc - it will perform the session data garbage collection

10. session_get_cookie_params - it will fetch the session cookie parameters

11.session_id - it will either get and/or set the current session module

12.session_is_registered - It will investigate whether a global variable is registered in a specific session

13.session_module_name - it will either get and/or set the current session module

14.session_name - it will either get and/or set the current session name

15. session_regenerate_id - it will update the current session id to a new one.

16.session_register_shutdown - session shutdown function

17.session_register - it will register 1 or more global variable using the current session you are using.

18.session_reset - it will reload session arrays with their original values

19.session_save_path - it will either get and/or set the current session save path

20. session_set_cookie_params - it will set the session cookie parameters

21. session_set_save_handler - it will sets the use-level session storage functions

22. session_start - it may start a new or resume an existing session

23. session_status - it will return the current session status

24. session_unregister - it will unregister a global variable from the current session

25.session_unset - it will free all session variables

26.session_write_close - it will write session data and end session

PHP is fast and flexible especially it is much powerful when it comes to a most well know sites in the world.

PHP is easy to use especially for those who are familiar with C, it’s very easy for them to learn and create a website using PHP.

It is not suitable for huge application so it's hard to maintain since it is not that modular and PHP is generally hard to expand the language to add a non-standard functionalities that isn’t handled by its built-in functions.

For security purposes since it's an open source everyone can view PHP's source code so it can easily hack it to identify bugs in the code.

PHP example codes

Creating a variable.

to declare a variable first type the dollar($) sign following it with the variable name along with the value.

$MidtermGrade = 65;

Echo and Print
Echo and Print is used to output the data on the screen. echo has no return value while print has a return value so that it can be used in expressions.

<?php

echo "HELLO world!"
echo "Hello Universe!"<br>
echo "This ", "string ", "was ", "made ", "with multiple parameters.";
?>


<?php
print "Hello world!<br>";

?>

PHP constants

Conditional Statements
if statement

<?php
$name = "miguel";
$grade = 95;

if ($name == "miguel") {
    echo "your grade is" . $grade . "<br>";
}
?>


if else statement

<?php
$name = "miguel";
$grade = 95;

if ($name == "miguel") {
    echo "your grade is" . $grade . "<br>";
}else
    echo "YOU ARE NOT MIGUEL.";
?>


if elsif statement

<?php
if ($name == "miguel") {
    echo "your grade is" . $grade . "<br>";
}elseif ($name != "miguel") {
  echo "YOU ARE NOT MIGUEL.";
}else 
  echo "PATATAS";
?>


Switch Case

<?php
switch ($laptopBrand) {
    case "asus":
        echo "Your laptop brand is Asus.";
        break;
    case "samsung":
        echo "Your laptop brand is Samsung.";
        break;
   default:
        echo "Your laptop brand is not Samsung or Asus.";

}
?>


While loop

<?php
$num = 0; 
while($num <= 3) {
    echo "number: $num";
    $num++;
?>


Do While loop

<?php
$num = 0;
do {
    echo "number: $num";
    $num++;
} while($num <= 3);
?>


For loop

<?php
for ($num = 0; $num <= 3; $num++) {
    echo "number: $num";
}
?>


Foreach loop

<?php
$num = array("0", "1", "2", "3"); 

foreach ($num as $digits) {
    echo "$digits";
}
?>


PHP user defined function

<?php

    echo "Hello universe.";
}
helloWorld();
?>
