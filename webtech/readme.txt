Steps on how to open our website:

Step 1: Extract the '9358A-G7' folder
Step 2: Open the 'WebtechFinale-master' folder. Then open the 'webtech' folder.
Step 3: Look for the 'login.html' file and double click it. Once you double click the file it will run on your default browser. Input your name so that you will be recognized in the website.
Step 4: You will now see the home page of our website. Drag your mouse pointer to "KNOW MORE ABOUT THE WEBSITE" to redirect you to the Final Notes page where the informations about the course websites are there. There you will see the different topics in finals in Web Systems and Technologies.
