What is AJAX?


AJAX(Asynchronous JavaScript and XML) is a technique that make our lives easier when creating a fast and dynamic web pages.
AJAX allows us to update web pages asynchronously by exchanging small amounts of information
with the server in the background.

Important Features of AJAX

AJAX make our web pages more faster, depends on server technology, increase the performance
of the web page and giving users an interactive web applications.
